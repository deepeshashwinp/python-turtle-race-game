# 🐢 Python Turtle Race Game

## 📖 Description
A Python Turtle graphics-based race betting game where players place a bet on a colored turtle and watch the race to see if their chosen turtle wins.

## ✨ Features
- Turtle race simulation
- User betting system
- Random race outcomes
- Color-based winner selection
- Built with Python Turtle Graphics

## 🛠️ Technologies Used
- Python
- Turtle Graphics
- Random Module

## ▶️ How to Run
1. Install Python 3.
2. Run the `main.py` file.
3. Enter the turtle color you want to bet on.
4. Watch the race and see if you win!
